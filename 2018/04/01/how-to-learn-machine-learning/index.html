如何学习 “机器学习”

想要学习机器学习首先要知道什么是机器学习，并且它和数据挖掘、深度学习有什么关系。

以下都是我个人在学习过程中的经验和观点，如有错误之处，还望见谅。

本文主要分为以下几个部分：

  1. 概念介绍，区分机器学习，数据挖掘，深度学习概念；
  2. 如何学习；
  3. 学习资源推荐；
  4. 应用。

1 What？是什么？

1.1 共同点

机器学习、数据挖掘、深度学习的功能是一样的，都是建立一个模型然后利用模型进行数据的分类和回归。

1.2 不同点

首先区分一下机器学习和数据挖掘的不同之处，其实这两个是在做相同的东西，但是这两个词的概念的侧重点不同。

  • 当我们讲机器学习的时候，我们更加侧重指的是机器学习算法，即重点指的算法。

  • 当我们讲数据挖掘的时候，我们更加侧重指的是如何从数据中挖掘出有效的信息，即数据是重点。

  • 深度学习是机器学习的子集，特指深度的神经网络。

虽然概念稍微有点差距，但是实际中，我们往往花在处理数据上的时间更多，为什么这样说呢？因为算法是前人开发出来的，往往我们很难去改进它，其实也没有这个能力，我们能做的就是从这些十几个机器学习算法中选择出适合我们当前数据使用的一个算法。

那既然我们不需要发明一个新的算法，那么为什么需要去学习算法并且最好要知道算法是如何实现的？因为算法有很多中种，并且他们的个性都不一样，适用于不同的数据，所以我们要根据数据和算法的个性来选择和匹配他们。

但是数据就不一样，不同的场景使用不同的数据，我们需要把数据处理成算法可以使用的形式，并且这是很花时间的。

  • 那什么是深度学习呢？其实现在讲的深度学习指的是层数非常多的神经网络，其实不必被深度学习这个词吓到，看起来很高大上，但是它比传统机器学习（也就是平时说的SVM，决策树）更加简单。深度学习不是一个理论性的东西（至少现在还不是），我值的是没有一个完整的理论来指出他为什么会这么好。学习深度学习往往是经验的积累，人们去做实验发现哪些网络结构会更好。没有什么概念推倒。

2 如何学习？

学习机器学习和学习其他方法一样（为什么我这里会使用机器学习这个词，因为我想特别指的是算法，毕竟如何挖掘数据在学习算法的过程中就自然会了）。

  • 动手（做实验）；
  • 看视频；
  • 看书；
  • 向他人请教和交流。

3 学习资源

  • 视频：推荐吴恩达的机器学习，和他的深度学习，下面给出链接：
    1. 吴恩达机器学习，bilibili上的：https://www.bilibili.com/video/av9559261
    2. 吴恩达深度学习，网易云课题：http://mooc.study.163.com/smartSpec/detail/1001319001.htm
    3. 李飞飞的计算机视觉：这个我还没看，但是非常有名的cs291：http://study.163.com/course/courseMain.htm?courseId=1003223001

  • 书：

    1. 算法：第一推荐：《统计学习方法》– 李航
    2. 数据挖掘：西瓜书：《机器学习》–周志华
    3. 深度学习花书：《深度学习》– Goodfellow
    4. 机器学习算法编程的书, 虽然是英文版的但是看起来难度也不是很大：《Hands On Machine Learning with Scikit Learn and TensorFlow》

  • 动手：

    1. 自己做实验是最重要的了，学了不用没多久也就完了，并且成功跑通一个算法也是非常有成就感的；
    2. 主要是通过调用别人写的算法的库来使用，调参。
    3. 也可以自己实现，可以参照《机器学习实战》这本书，有助于对算法的理解。
    4. 最后可以参加数据科学的比赛，主要网站有kaggle，天池。

先推荐看视频，然后看书，看书会陷入一个看不懂的状态，我觉得这个书的作者的表述能力也有关系，好的作者可以把很难的东西将明白。所以传统学习算法方面，推荐把《统计学习方法》的大部分看懂，李航还是很厉害的。

补充：

  • 数学基础方面，主要是求导，微分，概率论，矩阵相乘这方面的内容，
    1. 线性代数：http://study.163.com/course/courseMain.htm?courseId=1003649037
    2. 微积分：就看高数吧，用到再看，看不懂先跳过，对编程无影响，反正跑通。。。。
    3. 概率论：看课本把，学的时候看概念不多，还有李航的也讲的不错，需要学习的概念主要有，贝叶斯原理，最大似然估计，最大后验概率等等

注：中国的课程也出了非常多，这里主要指的是一些网上教育机构搞的网课，我当初不懂事也看了很多，国内的所有的课程都不推荐看（浪费时间），除非国外视频都看懂了以后无聊可以看看。

4 应用

我认为机器学习、人工智能目前主要使用的地方有两个（可以商业应用起来的），计算机视觉（对图像识别）和自然语言处理（文本啊，语音）。其他有智能推荐算法，高利贷领域的风控等等（这些就不单单是使用机器学习了）。

并且在计算机视觉这一块传统的机器学习几乎已经无用武之地了。自然语言处理我接触的比较少，但貌似传统机器学习还是有用的。

传统机器学习的弱点在于需要人工构建特征，并且准确率方面弱于深度学习。

深度学习是大量数据驱动，但是网络的框架也是非常重要的，没有框架数据也是一无所用。

5 几句话总结

  • 传统机器学习虽然没落，但是有必要去学习，学习以后有助于深度学习的理解，和对数据的理解。
  • 深度学习主要侧重于图像和自然语言处理。
  • 编程，可以直接使用开源框架毕竟自己写算法很无聊，有些功能根本无法实现（以后有能力可以自己写），深度学习主推TensorFlow，传统机器学习主推scikit-learn。
  • 使用python。

祝好。。。
