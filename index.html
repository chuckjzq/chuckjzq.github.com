注：JSON格式是字符串。
JSON(JavaScript Object Notation, JS 对象标记) 是一种轻量级的数据交换格式。
简单地说，JSON 可以将 JavaScript 对象中表示的一组数据转换为字符串，然后就可以在网络或者程序之间轻松地传递这个字符串，并在需要的时候将它还原为各编程语言所支持的数据格式。
