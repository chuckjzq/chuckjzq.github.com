spark入门学习

安装

下载地址：https://spark.apache.org/downloads.html
可以下载对应Hadoop版本的spark，但是spark本身是不依赖Hadoop的。

  • pyspark shell: bin/pyspark
  • scala shell: bin/spark-shell

测试

读取文本，并计算行数等。

修改日志级别

spark目录下的conf目录下的log4j.properties.template文件改名成cp log4j.properties.template log4j.properties使之生效。
然后更改成以下内容：
log4j.rootCategory=WARN, console

Scala安装

安装scala和idea

  • 从 Scala 官网地址 http://www.scala-lang.org/downloads 下载 Scala 二进制包, jdk需要1.5以上版本。
  • 可以把二进制文件解压到/usr/local/share下，配置path，~/.bash_profile,然后source ~/.bash_profile

配置ssh无密登录

  • ssh-keygen
  • .ssh目录下 cat xxx_rsa.pub > suthorized_keys
  • chmod 600 authoried_keys

运行spark作业

  • 启动master
    • ./sbin/start-master.sh
  • 启动worker
    • ./bin/spark-class org.apache.spark.deploy.worker.Worker spark://localhost.localdomain:7077
  • 提交作业
    • ./bin/spark-submit --master spark://localhost.localdomain:7077 --class WordCount /home/maixia/soft/pro.jar

RDDs介绍

  • Driver program：
    • 包含程序的main()方法，RDDs的定义和操作。
    • 它管理很多节点，称之为executors。
  • SparkContext:
    • Driver programs 通过SparkContext对象访问Spark。
    • SparkContext 对象代表和一个集群的连接。
    • 在Shell中SparkContext 自动创建好了，就是sc。
  • RDDs：
    • 弹性分布式数据集，就是一个数据集。
    • 并行的分布在集群中的，如果一个数据集很大，那么就会自动的分发到多台机器上。
    • RDDs是Spark分发数据和计算的基础抽象类。分发就是把整个大文件切成一些小文件，分给不同的机器。
    • 操作是在整体的数据上操作，而不是对分发的小数据集操作。
    • 一个RDD是一个不可改变的分布式集合对象。
    • Spark中，所有的计算都是通过RDDs的创建，转换，操作完成的。如上面例子的lines对象。
    • 一个RDD内部由许多partitions(分片)组成。
  • 分片：
    • 每个分片包括一部分数据，Partitions可在集群不同节点上计算。
    • 分片Spark并行处理的单元，Spark顺序的，并行的处理分片。

RDDs的创建

  • 通过SparkContext的parallelize函数来创建
  • 通过外部文件创建：scala> val lines = sc.textFile(“../helloSpark”)
