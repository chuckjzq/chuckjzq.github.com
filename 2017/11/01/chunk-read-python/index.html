通过pandas分块读取文件

有时候在数据量非常大的时候，无法一次读取这么大的数据到内存中处理，第一可能内存不够，第二读取速度较慢，处理速度较慢。

解决方法

  • 可以通过每次读取一定行数的数据来处理。
  • 如果在内存足够的情况下要读取整个文件的数据，那么也可以分块读取后再拼接。

pandas实现

  • 通过指定参数iterator=True，然后使用get_chunk(行数)，每次调用get_chunk都会读取下一行数据。
  • 通过指定参数chunksize=4直接读取，返回的是一个迭代器，可以使用for循环迭代读取。
