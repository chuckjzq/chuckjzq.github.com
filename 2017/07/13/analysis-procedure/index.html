数据分析步骤

Jul 13, 2017 | 数据分析 | 阅读
  • 明确问题
  • 问题分解
  • 做出假设
  • 验证假设
  • 做出决策
  • 提出疑问

  • 循环上述步骤

    确定问题

    首先，要明确问题。
    通过客户了解信息，明确问题。对于数据分析这来说，客户是分析结果的服务对象，有可能就是你老板。
    通过提出问题来明确问题！

  • 量化目标数据。

  • 对某些数据感到好奇。
  • 预见客户的想法。

把问题和数据分解

  • 把大问题分解为小问题: 把模糊的大问题，分解为具体的小问题。
  • 把数据分解为更小的组块: 提炼重要因子。
  • 针对问题的基准假设

分析数据数值的大小和趋势，与结果的相关性。

评估组块

评估分解的组块。针对提出的小问题，然后评估自己的假设。

分析从你介入的那一刻开始

让自己介入分析的意思是作出自己的明确假设，并且以自己的信用为自己的结论打赌。

提出建议

作为数据分析师，就是要让自己和客户仔细研究你对数据的评估，洞察先机，从而作出更好的决策。
将自己的设想和判断以合适的格式整合起来，供客户选择。
但要简单，但也要传达到位。

除非将分析形成报表供制定决策，否则，分析将毫无用处。
报表格式：

  1. 背景
  2. 数据解说
  3. 建议

不确定的因素

思考自己不知道的地方，指出不确定的因素。发现盲点。

让客户思考问题

胆子要大，只有向客户提出问题，才能更好的分析我们的问题。

总结：反复思考问题，分解问题，明确问题，假设目标，提出解决方案。

控制组

要知道方案是否可以行，有时候不能只和以前的数据比较，因为没有对照组无法真正的说明情况。
