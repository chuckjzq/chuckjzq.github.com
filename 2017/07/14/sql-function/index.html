函数的类型

在 SQL 中，基本的函数类型和种类有若干种。函数的基本类型是：

  • Aggregate 合计函数：Aggregate 函数的操作面向一系列的值，并返回一个单一的值。
  • Scalar 纯量函数：Scalar 函数的操作面向某个单一的值，并返回基于输入值的一个单一的值。

GROUP BY 分组语句

GROUP BY 语句用于结合合计函数，根据一个或多个列对结果集进行分组。
例如可以对人进行分组，再使用sum函数，对每个人计算他的工资。
