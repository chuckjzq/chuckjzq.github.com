sklearn简单介绍

Jul 7, 2017 | sklearn | 阅读

sklearn 作为一个机器学习工具，那么他的主要功能就是建立模型和预测。

假设函数 + 拟合参数 + 预测

假设函数， 假设函数.fit()， 假设函数.predict()

选择假设函数（Hypothesis function）

当然sklearn里面不把模型称为假设函数，它称之为estimator。 我还是喜欢称之为假设函数。

sklearn中包括了很多常用的模型，例如有svm

这样我们就可以设置一个名字为clf的假设函数，并且选择参数gamma和C 。
当然也可以先直接建立一个没有参数的假设函数，然后再设置参数，然后直接拟合参数。并且这样可以在后面的过程中更改设置参数。

拟合模型参数

sklearn中的所有模型都已经实现了fit(X, y)方法和 predict(X)方法。
当我们有了假设函数以后，接下来就是拟合参数sklearn里面的svm好像是直接fit()函数搞定。不知道可不可以自己设置损失函数，然后在使用优化算法例如梯度下降去拟合参数。这个后面再看回来再补充。。。。。。

预测

使用predict()函数

保存模型

可以有两种方法：

  • 可以使用 pickle: dumps 和 loads 方法
  • 可以使用 joblib: dump 和 load 方法

官方文档中说如果数据量大的情况下使用joblib

Note that pickle has some security and maintainability issues.

约定

sklearn中要遵守的一些约定。

数据类型

数据类型一定要是float64类型。
额可以使用transformer.fit_transform(X)转换数据类型。
X_new = transformer.fit_transform(X)

如果label是数字的话，那么输出就是数字，如果label是字符串的话，预测的输出也是字符串，相当于输出和输出的时候映射了一下。

多分类和多标签拟合

  • 多分类(multiclass): 多分类，但是每个数据只属于一个类别
  • 多标签(multilabel): 多分类，但是每个数据可能不止属于一个类别
