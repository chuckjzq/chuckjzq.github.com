sklearn-实现随机梯度下降

Stochastic Gradient Descent (SGD)随机梯度下降。
在sklearn中可以使用SGDClassifier 和 SGDRegressor类，第一个是分类，第二个是回归。

partial_fit为minibatch拟合。

使用SGDClassifier

class sklearn.linear_model.SGDClassifier(loss=’hinge’, penalty=’l2’, alpha=0.0001, l1_ratio=0.15, fit_intercept=True, max_iter=None, tol=None, shuffle=True, verbose=0, epsilon=0.1, n_jobs=1, random_state=None, learning_rate=’optimal’, eta0=0.0, power_t=0.5, class_weight=None, warm_start=False, average=False, n_iter=None)

  • loss参数，选择损失函数，’hinge’为SVM,’log’为逻辑回归。
  • penalty参数，惩罚项
  • alpha：Constant that multiplies the regularization term. Defaults to 0.0001 Also used to compute learning_rate when set to ‘optimal’.
  • learning_rate参数，学习率
    • ‘constant’: eta = eta0
    • ‘optimal’: eta = 1.0 / (alpha * (t + t0)) [default]
    • ‘invscaling’: eta = eta0 / pow(t, power_t)
  • n_jobs：The number of CPUs to use to do the OVA (One Versus All, for multi-class problems) computation. -1 means ‘all CPUs’. Defaults to 1.
  • shuffle=True,打乱训练数据
