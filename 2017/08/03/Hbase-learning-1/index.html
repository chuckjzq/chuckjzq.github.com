Hbase初窥笔记

HBase是一个Nosql（not only sql 不仅仅是用sql来操作，例如用put，get）数据库，列式存数的数据库，多版本。

数据按照时间分常有这些：

  • 热数据
  • 懒数据
  • 冷数据

Hbase场景：海量数据，实时查询，交通数据，账单数据，游戏数据，电商交易数据。

  • 开源
  • 分布式
  • 多版本

  • 非关系型

    HBase 功能

  • Hadoop 数据库

    • 存储数据
    • 检索数据
    • 海量数据
      • 数据条目数上亿
    • 检索的速度：准实时、秒级别
    • 基于HDFS
      • hdfs优势：
        • 1）数据的安全性（三个副本，检测损坏）
        • 2）普通商用pc sever（内存大点，磁盘大点）

框架

依赖zookeeper，
有Hmaster，HRegionserver，NameNode，DataNode

按列存储

每条数据有唯一的标识符：
rowkey - 主键
面向列的数据库
rowkey + columnfamily + column01 + timestamp: value => cell (面向列的单元)

例如：

HBase表的数据，可以快速查询，关键点在于表中的主键设计。

Hbase只要知道表名和列名，都是以字节数组存储byte[]。
bytes

表的结构


  • Row Key
    • Byte array
    • 表中每条记录的主键
    • 方便快速查找
  • column Family
    • 拥有一个名称（string）
    • 包含一个或者多个相关列
  • Column
    • 属于某一个column family
    • 包含在某一列中
      • familyName:columnName
  • Version Number
    • 每个rowkey唯一
    • 默认值-》系统时间戳
    • 类型为long
  • value（cell）
    • byte array

HBase体系结构

ZooKeeper是一个分布式的，开放源码的分布式应用程序协调服务，是Google的Chubby一个开源的实现，是Hadoop和Hbase的重要组件。它是一个为分布式应用提供一致性服务的软件，提供的功能包括：配置维护、域名服务、分布式同步、组服务等。

创建表

可以使用命令创建一个表，在这里必须指定表名和列族名。在HBase shell中创建表的语法如下所示。

create ‘<table name>’,’<column family>’

查看表信息

describe tabel_name

插入数据

HBase中只有插数据和查询数据，没有更新数据这个东西(这个老师讲的不一定对。。。)，因为列有“版本Version这个东西”。

可以使用put命令更新现有的单元格值。按照下面的语法，并注明新值，如下图所示。
put ‘table name’,’row ’,'Column family:column name',’new value’
新给定值替换现有的值，并更新该行。
注：从这里可以发现，插入的方式不是一下子插入一行，而是一个个单元插入，即列。

HBase的数据查询的三种方式

  • 依据rowkey查询，最快的：get
    下面两个可以限定查询的列。
  • 范围查询：scan range
  • 全表扫描：scan

get

get指定行

get命令和HTable类的get()方法用于从HBase表中读取数据。使用 get 命令，可以同时获取一行数据。它的语法如下：

get ’<table name>’,’row1’

get指定列

下面给出的是语法，使用get方法读取指定列。

hbase>get 'table name', ‘rowid’, {COLUMN => ‘column family:column name ’}
注：上面大括号中的就是条件。

scan range

scan 命令用于查看HTable数据。使用 scan 命令可以得到表中的数据。它的语法如下：

scan ‘<table name>’ + 条件

还可以指定过滤条件，类似于sql中的where。

scan

scan ‘<table name>’

删除

从表删除特定单元格

使用 delete 命令，可以在一个表中删除特定单元格。 delete 命令的语法如下：
delete ‘<table name>’, ‘<row>’, ‘<column name >’, ‘<time stamp>’

删除表的所有单元格

使用“deleteall”命令，可以删除一行中所有单元格。下面给出是 deleteall 命令的语法。

deleteall ‘<table name>’, ‘<row>’,
