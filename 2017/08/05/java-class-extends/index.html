面向对象三大特性 之继承

类下面的变量称之为属性，方法中的变量称之为变量。

继承

  1. 概念：
    • 继承是类与类之间的关系，是一种 “is a” 的关系。
    • Java中是单继承
  2. 继承的好处
    • 可以少写很多代码：子类用于父类的所有的属性和方法
    • 代码的复用
  3. 语法规则：class 子类 extends 父类 { }

方法的重写

  1. 什么是方法的重写：当子类对父类的方法不满意时可以重写父类的方法，在调用方法时优先使用子类的方法。

  2. 语法规则

    • 放回值类型
    • 方法名

    • 参数类型和个数

      都要和父类继承的方法相同，这才叫方法的重写。

final 关键字

使用final做修饰词有“最终的”意思。

final可以修饰类、方法、属性和变量。（这里属性和变量，属性值的是类下面的属性，变量指的是方法里面的变量）。

  • 修饰类，该类不允许被继承。
  • 修饰方法，该方法不允许被重写。
  • 修饰属性，该属性不会被隐式（默认值为0）初始化，类的初始化属性必须有值，或者在构造函数中初始化。
  • 修饰变量，该变量只能被赋值一次，称之为常量（最好字母全大写）。

super 关键字

在对象的内部使用，表示代表父类对象。（这里对象和类也不一样，但哪里不一样，只可意会。。。）

  1. 访问父类属性：super.属性
  2. 访问父类方法：super.方法()，可以在子类的方法中调用父类方法。
  3. 子类的构造过程当中必须调用父类的构造方法，必须在第一行。隐式会默认调用父类的无参方法。

  1. 如果子类中没有显示的调用父类的构造方法，而父类中又没有无参构造方法（默认时无无参构造方法可调），则编译出错。

Object类

Object类是所有类的父类，如果没有extends继承某个类，则默认继承Object类。

Object类中的方法，适合所有子类。

  • toString方法：在Object类中定义toString的时候返回对象的哈希码（通过哈希算法算出来的，区分每一对象，对象地址字符串）。
  • 可以通过重写toString()方法表示对象的属性。
  • equals()方法：比较的是对象的引用是否指向同一块内存地址。
