采集JavaScript

  • JavaScript是客户端脚本语言，它是一种运行在浏览器而非服务器上的语言。
  • 它可以收集用户的跟踪数据，不需要重载页面直接提交表单，在页面中嵌入多媒体文件。
  • 在<script>标签中

JavaScript简介

  • 弱类型语言
  • 脚本形式（有点像python）
  • 把函数当做变量使用

常用的JavaScript库

  • jQuery：可以动态的创建HTML内容，只有在JavaScript代码执行后才会显示，传统方法无法采集。
  • Google Analytics：采集的时候要关掉那些分析工具的cookie（目前不知道干嘛用）
  • Google地图：python可以获得google.maps.Latlng()里面的坐标。

Ajax和动态HTML

如果提交表单后，页面不需要刷新则是用了Ajax技术，它是一系列的技术。（异步JavaScript和XML）

与Ajax一样，动态HTML（dynamic HTML）也是一系列用于解决网络问题的技术集合。DHTML是用客户端语言改变页面的HTML元素。

两种途径抓取：

  • 直接从JavaScript代码里采集内容；
  • 或者用python的第三方库运行JavaScript，直接采集你在浏览器里看到的页面。

工具

Selenium + PhantomJS

Selenium: 是一个网络数据采集工具，可以让浏览器自动加载页面，获取需要的数据，或者判断段网站上某个动作是否发生。
PhantomJS: 无头浏览器，没有界面的浏览器，和Selenium配合使用。

其他工具

  • XPath （XML Path）是一种查询语言，在xml文档中导航和选择元素。
  • DOM 可以以一种独立于平台和语言的方式访问和修改一个文档的内容和结构，总之可以加到js中产生一些动作。
